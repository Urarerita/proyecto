Sitio Web
Un sitio web es un conjunto de páginas web relacionadas que están almacenadas en un servidor y accesibles a través de internet mediante una URL. Puede contener información, imágenes, videos, formularios y otros elementos interactivos. Ejemplo: Amazon, Wikipedia, YouTube.
Página Web
Es un documento digital escrito en HTML que forma parte de un sitio web. Puede ser estática (contenido fijo) o dinámica (contenido que cambia según la interacción del usuario).
Hipervinculo
También llamado enlace, es un elemento que permite saltar de una página a otra dentro de un sitio web o a otro sitio en internet. Se representa con la etiqueta en HTML.
URL (Uniform Resourse Locator)
Es la dirección única que identifica un recurso en internet. Está compuesta por el protocolo (https://), el dominio (www.ejemplo.com) y la ruta del recurso (/pagina).
